3. 표준 입출력

입출력 함수는 표준 입출력, 파일 입출력, 저급 입출력 함수가 있다.
표준 입출력 함수는 키보드와 모니터를 통해서 입출력하는 함수이고,
파일 입출력 함수는 파일을 통해서 자료를 입출력하는 함수이며
저급 입출력 함수는 os내의 시스템 함수를 통해서 자료를 입출력하는 함수라는데
솔직히 무슨 뜻인지 정확히 이해되지는 않는다.

표준 입출력 함수는 형식화된 입출력, 문자 입출력, 문자열 입출력 함수가 있다.

형식화된 입출력
%d 10진수 부호있는 정수
%o 8진수
%x 16진수
%u 부호없는 정수
%e 부동소수 과학값
%f 부동소수 실수
%g %e, %f중 작은 거
%c 문자
%s 문자열
%n %n이 나타날 때까지 출력한 문자 수
%ld 긴 부호있는 정수
%lu 긴 부호없는 정수

당연하지만 scanf는 포인터를 입력으로 받아서 값을 때려박는 친구이다

\a 경고음
\t 탭
\n 다음줄로
\r 현재 줄의 처음으로
\" "
\' ' 
\\ \
%% %

문자 입출력과 문자열 입출력
//이게 표준 입출력의 마지막 파트여서 오늘은 코드 문제들도 몇개정도 풀어볼까... 생각중이다.

int getchar() //문자 하나를 입력받는 함수
return ascii

int putchar(char x) //x를 출력하는 함수(절대 배열을 넣으면 안 된다.)

char * gets(char *str) //문자열을 입력하는 함수. 당연하게도, scanf처럼 포인터 값을 입력받아서 콜 바이 레퍼런스로 값을 추가시킨다.
gets함수는 scanf와 다르게 띄어쓰기 또한 입력한다. (scanf에서는 \0으로 간주한다.)

char * puts(char *str)
char가 배열이기에 포인터를 받는 것 같으며, 그 값을 출력하는 역할을 한다.
